NEXORA AI — INTELLIGENT AI WORKSPACE
=====================================

PROJECT OVERVIEW
----------------

Nexora AI is a modern AI-powered intelligent workspace built using Python and Streamlit.

The project transforms a basic chatbot into an interactive AI assistant that provides intelligent responses through a modern conversational interface.

The application is designed to provide a professional AI chat experience with conversation memory, multiple assistant modes, customizable response preferences, prompt enhancement, chat management, and conversation export features.

MAIN FEATURES
-------------

1. MODERN AI CHAT INTERFACE

Nexora AI uses Streamlit's conversational components to provide a modern chat experience.

Main components:

st.chat_input()
    |
    v
User Message
    |
    v
AI Processing
    |
    v
st.chat_message()
    |
    v
AI Response


2. CONVERSATION MEMORY

The application uses Streamlit Session State to maintain conversation context.

st.session_state
    |
    +----> User Messages
    |
    +----> AI Responses
    |
    +----> Current Conversation

This allows users to ask follow-up questions during the same session.


3. AI PERSONALITY MODES

Users can select different AI assistant modes depending on their requirements.

Available modes may include:

- General Assistant
- Coding Expert
- Study Tutor
- Research Assistant
- Creative Writer
- Career Advisor

Each mode changes the instructions provided to the AI model.


4. RESPONSE CUSTOMIZATION

Users can customize how Nexora AI responds.

Response Length:

- Short
- Medium
- Detailed

Response Tone:

- Professional
- Friendly
- Simple


5. PROMPT ENHANCEMENT

Nexora AI can improve short or unclear prompts before processing them.

Example:

User Prompt:
"Explain ML"

Enhanced Prompt:
"Explain Machine Learning in simple language with its major types and real-world examples."

The original intention of the user is preserved.


6. CHAT MANAGEMENT

The application provides chat management features such as:

- Start New Chat
- Clear Current Conversation
- Maintain Current Session History
- View Chat Statistics


7. SMART SUGGESTIONS

When the conversation is empty, Nexora AI can display useful prompt suggestions.

Examples:

- Explain a complex topic
- Help me write code
- Summarize something
- Brainstorm project ideas


8. STREAMING AI RESPONSES

AI responses are displayed progressively to provide a modern conversational experience similar to popular AI assistants.


9. CONVERSATION EXPORT

Users can export their current conversations for future use.

Supported formats may include:

- TXT
- Markdown


10. CHAT STATISTICS

The sidebar can display session information such as:

- Number of User Messages
- Number of AI Responses
- Total Messages

LOCAL PROJECT SETUP
-------------------

Step 1: Open the project folder.

Step 2: Create a virtual environment.

python -m venv .venv


Step 3: Activate the virtual environment.

Windows PowerShell:

.\.venv\Scripts\Activate.ps1


Step 4: Install required dependencies.

pip install -r requirements.txt


Step 5: Create a .env file if the application requires API credentials.

Example:

API_KEY=your_api_key_here

IMPORTANT:

Never upload the .env file to GitHub.

Make sure .gitignore contains:

.env
.venv/
__pycache__/


Step 6: Run the application.

streamlit run app.py

OR

python -m streamlit run app.py

SECURITY
--------

- Never hardcode API keys in Python files.
- Store API credentials inside the .env file.
- Never push the .env file to GitHub.
- Keep sensitive credentials inside environment variables.
- Revoke and regenerate any API key that is accidentally exposed.
