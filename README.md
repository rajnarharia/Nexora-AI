Welcome to **Nexora AI** — an ultra-premium, highly customizable AI assistant built for professionals, students, and creators. By leveraging the immense speed of the Groq API and a state-of-the-art glassmorphism UI, Nexora transforms standard chat interfaces into a true neural workspace.

## 🌟 Stunning Features

| Feature | Description |
| :--- | :--- |
| 💎 **Ultra-Premium UI** | High-end glassmorphism styling, animated gradients, blur effects, and deep dark-mode aesthetics. |
| 🧠 **Dynamic Neural Engines** | Seamlessly switch between **Llama 3 (70B)**, **Mixtral (8x7B)**, and **Gemma (7B)** instantly. |
| 🎭 **AI Personas** | Dedicated modes: *Coding Expert, Study Tutor, Data Analyst, Creative Writer, & Career Advisor*. |
| 🎛️ **Granular Output Tuning** | Precision control over response length, tonal delivery, and maximum token output limits. |
| ✨ **Neural Prompt Enhancer** | Built-in AI middleware that automatically optimizes and refines your prompts before execution. |
| 🔮 **Smart Logic Paths** | Anticipates your needs by generating 3 highly contextual follow-up questions after every response. |
| 📈 **Advanced Analytics** | Real-time session monitoring of message counts and estimated token consumption. |

## 🛠️ Architecture & Tech Stack

- **Frontend Interface:** [Streamlit](https://streamlit.io/) with custom CSS injections.
- **Inference Engine:** [Groq API](https://groq.com/) for near-instantaneous token generation.
- **Environment Handling:** `python-dotenv` for secure secret management.

## ⚙️ Quickstart Installation

Follow these instructions to deploy your own Nexora AI core locally.

**1. Clone the repository**
```bash
git clone https://github.com/rajnarharia/Nexora-AI.git
cd Nexora-AI
```

**2. Initialize Virtual Environment**
```bash
python -m venv .venv
source .venv/bin/activate  # Windows: .venv\Scripts\activate
```

**3. Install Core Dependencies**
```bash
pip install -r requirements.txt
```

**4. Configure Neural Keys**
Create a `.env` file in the root directory and securely paste your Groq API Key:
```env
GROQ_API_KEY=your_actual_groq_api_key_here
```

**5. Ignite the Core**
```bash
streamlit run app.py
```
*The interface will automatically deploy to your browser at `http://localhost:8501`.*
